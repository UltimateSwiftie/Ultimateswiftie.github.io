Interactive Flashcards

What is HTML?
HTML stands for HyperText Markup Language, which is used to create the structure of web pages.
What is CSS?
CSS stands for Cascading Style Sheets, which is used to control the presentation of web pages.
What is JavaScript?
JavaScript is a programming language that enables interactive elements on web pages.
What is Responsive Web Design?
Responsive web design ensures web pages look good on all devices by using fluid grids, flexible images, and media queries.
What is SEO?
SEO stands for Search Engine Optimization, which aims to improve a website's visibility in search engine results.
What is a Framework?
A framework is a standardized set of concepts, practices, and criteria for dealing with a common type of problem.
What is Git?
Git is a distributed version control system for tracking changes in source code during software development.
What is a Database?
A database is an organized collection of data, typically stored and accessed electronically from a computer system.
